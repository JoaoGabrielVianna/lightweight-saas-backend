# 🧩 Lightweight SaaS Backend (Go)

> **Status: `v0.0.0` — Early Development (BETA)**

A simple backend foundation for building SaaS applications with Go.

## 🚀 Overview

This project aims to provide a **clean and practical backend structure** for SaaS products.

The focus is on implementing only what is essential:

- authentication  
- multi-tenant architecture  
- a real business module (lead management)  

No overengineering. Just a solid foundation.

## 🎯 Goal

To build a backend that:

- is easy to understand  
- is fast to extend  
- reflects real-world SaaS needs  

## 📦 Current Status

🚧 This project is under active development.

Core features are being implemented step by step, starting with authentication and multi-tenant support.

## 💡 Purpose

This project is part of my journey to build production-ready backend systems and work with international teams.
